Inject following variables:
  • NODE_OWNER - UserId of the node owner
  • NODE_OWNER_EMAIL - E-mail of the node owner
  • NODE_COOWNERS - Comma-separated list of node co-owners (includes owner as well)
  • NODE_COOWNERS_EMAILS - Comma-separated list of e-mails of node co-owners (includes owner's e-mail)
All variables may be empty if there's no ownership info available.
