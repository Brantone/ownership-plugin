E-mail body template for "Contact Service Owners" link.

The entry supports the variables substitution (see the list below). Missing variables won't be substituted.

  • USER_ID - ID of the current user
  • USER_FULL_NAME - Full name of the current user
  • ITEM_TYPE_NAME - Type name of the currently item (job, node, build, etc.)
  • ITEM_DISPLAY_NAME - Display name of the item
  • ITEM_URL - Absolute URL to the item (if exists)
